# Aim

learn how to use gulp

# Details

* initialze project

```
$ yarn init -yp
$ yarn add gulp gulp-sass --dev
```

* (Optional) add .gitignore

```
$ echo "node_modules" >> .gitignore
$ echo "dist" >> .gitignore
```

* started

```
$ vim gulpfile.js
$ mkdir css
$ vim css/style.scss
$ vim package.json
$ yarn run build
```
